# 📚 기술 질문 구조화 채팅 내용 정리

## 🧭 대화 목적
- 기술 질문을 GPT에게 더 잘 전달할 수 있도록 **구조화된 질문 생성 방식**을 정립
- 두서없이 들어온 사용자 입력을 **논리적이고 체계적인 질문**으로 바꾸는 흐름 설계
- GPT에게 요구할 **역할, 출력 형식, 예시, 템플릿** 등을 정의하고 문서화

---

## 📌 핵심 흐름 정리

### 1. 역할 및 목표 정의
- GPT 역할: **시니어 소프트웨어 엔지니어 + 기술 프롬프트 전문가**
- 목적: 사용자 입력을 구조화된 고품질 기술 질문으로 변환

### 2. 작업 절차 설계
- 사용자 입력 요약
- 질문 정리 결과 출력
  1. 📂 유형 분류
  2. 🧾 사전 설명
  3. ❓ 질문 항목
  4. 📤 GPT용 최종 질문 문장
- 정보 부족 시 인터뷰식 추가 질문 (이유 포함)

### 3. 출력 형식 정리
- 줄바꿈, 강조, 이모지 등 시각적으로 명확한 구조 사용
- 마크다운(.md) 파일 포맷으로도 출력 가능

### 4. 실전 예시 생성
- JWT 인증 오류를 주제로 예시 작성
- GPT가 이해한 내용 → 질문 정리 결과 → 최종 질문 문장으로 구성
- 이를 기반으로 마크다운 파일 생성

### 5. 문서화 및 결과물 생성
- 전체 가이드를 하나의 마크다운 문서로 저장 (`Structured-tech-question-guide`)
- 실전 예시는 별도 `.md` 파일로 제공
