# 아이폰 단축어: 줄 단위 메모 생성 자동화

## 🎯 목적
복사한 텍스트를 줄 단위로 나누어 iCloud 메모에 각각 다른 메모로 저장하되, **사용자가 입력한 폴더명으로 메모를 관리**하는 단축어를 만든다.

---

## ✅ 기능 요약
- 클립보드 텍스트 불러오기
- 줄바꿈 기준으로 텍스트 분리
- 오늘 날짜(`2025년 6월 8일`)를 예시로 사용자에게 폴더명을 입력받음
- 입력한 폴더명으로 iCloud 메모 앱 내 폴더 선택
- 줄마다 새 메모를 해당 폴더에 저장

---

## 🧩 단축어 단계별 구성

### 1. 클립보드에서 텍스트 가져오기
- 작업: **클립보드에서 텍스트 가져오기**

### 2. 텍스트 분리
- 작업: **텍스트 분리**
  - 구분자: 줄바꿈 (`\n`)

### 3. 현재 날짜 구하기
- 작업: **현재 날짜 가져오기**
- 작업: **날짜 형식 지정**
  - 형식: `yyyy년 M월 d일`

### 4. 사용자에게 폴더명 입력 요청
- 작업: **질문 표시**
  - 질문: `"저장할 메모 폴더 이름을 입력하세요"`
  - 기본 응답: 위에서 만든 날짜 예시값 (`2025년 6월 8일`)

### 5. 반복 (분리된 각 줄마다)
- 작업: **반복**
  - 항목: 텍스트 분리 결과

#### 반복 내부 작업
- 작업: **새 메모 만들기**
  - 내용: 반복 항목
  - 폴더: 사용자 입력 폴더명
  - 계정: **iCloud**

---

## 📌 참고 사항
- "폴더 생성"은 단축어 앱이 직접 수행할 수는 없지만, **지정한 이름의 폴더가 없으면 iCloud 메모 앱에서 자동 생성됨**
- 해당 폴더가 이미 있다면 그 폴더에 저장됨

---

## 📝 예시 흐름

```
복사한 텍스트:
할 일 목록
- 우유 사기
- 운동하기

사용자 입력 폴더명: 2025년 6월 8일

→ 생성된 메모:
1. "할 일 목록" (in 폴더)
2. "- 우유 사기" (in 폴더)
3. "- 운동하기" (in 폴더)
```

- 메모 태그 붙이기
