아이폰에서 “한 줄씩 나뉜 메모 내용을 각각 새로운 메모로 만들고 싶을 때” 아래 방법 중에서 선택할 수 있어:

---

### ✅ 방법 1: **단축어 앱 이용 (자동화)**

1. **단축어 앱 열기**
   → iOS에 기본 설치되어 있음

2. **단축어 새로 만들기**

   * * 버튼 눌러 새 단축어 생성
   * 작업 추가 → “텍스트” 선택

     * 복사해온 줄바꿈 포함된 메모 붙여넣기
   * 작업 추가 → “텍스트 분리”

     * "줄바꿈"으로 분리 (또는 "줄바꿈"이 기본)
   * 작업 추가 → “반복”

     * 각 줄마다 반복
     * 반복 안에서 “새 메모 만들기” 추가

3. **단축어 실행**
   → 복사한 텍스트 줄마다 각각 메모 생성됨 (기본 메모 앱 기준)

> 🔁 반복 예시 흐름
> `[텍스트] → [텍스트 분리 (줄바꿈)] → [반복] → [메모 만들기]`

---

### ✅ 방법 2: **수동으로 복사 붙여넣기**

단축어 설정이 어렵거나 원하지 않는다면:

1. 메모 앱에서 내용을 줄 단위로 복사
2. 하나씩 메모 앱에 새 메모로 붙여넣기
   → 다소 번거롭지만 앱 없이 가능

---

### ✅ 방법 3: **단축어 예시 공유 (원한다면 설치 파일 제공 가능)**

원한다면 내가 단축어 공유 링크나 `.shortcut` 파일을 만들어 줄게.
복사한 텍스트를 넣으면 자동으로 줄마다 새 메모가 생성되게 할 수 있어.

---

필요한 방식 골라줘.
예:

* 단축어 방식 원하면 → “단축어로 만들어줘”
* 수동만 가능하면 → “수동으로 할게”
* 공유 링크 원하면 → “단축어 공유해줘”

## 📱 아이폰 단축어: **"줄 단위 새 메모 만들기" 만들기**

### ① 단축어 앱 실행

* 아이폰에서 **"단축어" 앱** 실행
* 우측 상단 **`+` 버튼** 눌러 새 단축어 생성

---

### ② 아래 작업 순서대로 추가

#### ✅ 1. \[텍스트] 추가

* **작업 추가** → `텍스트` 선택
* 예시 내용 입력 or 나중에 공유시 복사한 내용 자동 사용 가능

```plaintext
첫 번째 메모
두 번째 메모
세 번째 메모
```

👉 나중에 복사한 내용을 넣고 싶으면, 텍스트 입력 대신 "클립보드에서 가져오기"도 가능함. (필요 시 따로 알려줄게)

---

#### ✅ 2. \[텍스트 분리하기] 추가

* **작업 추가** → `텍스트 분리` 또는 `Split Text` 선택
* **입력:** 위 텍스트
* **구분자:** "줄 바꿈"

> 이 작업은 텍스트를 줄마다 나누는 역할

---

#### ✅ 3. \[반복] 추가

* **작업 추가** → `반복` 또는 `Repeat with each` 선택
* 반복 대상: 앞 단계에서 분리된 항목

> 각 줄마다 반복 작업을 수행

---

#### ✅ 4. \[메모 만들기] 추가 (반복 안에 넣기)

* 반복 블록 안에서 작업 추가 → `새 메모 만들기` 또는 `Create Note` 선택
* **내용:** `반복 항목` (자동으로 나옴)

> 각 줄을 새로운 메모로 저장

---

## ✅ 2단계: 사용자에게 텍스트 보여주고 수정시키기

* 🔍 검색창에 `텍스트 입력 요청` 또는 `입력 요청`

  * **질문:** `"메모로 만들기 전 내용을 수정하세요 (줄마다 한 줄씩 메모 생성됩니다)"`
  * **기본 응답:** → `클립보드에서 가져온 텍스트` 연결

> 이 단계에서 사용자가 복사한 텍스트를 수정 가능!

---

## ✅ 3단계: 텍스트 분리 (줄 단위)

* 🔍 검색창에 `텍스트 분리` 입력
* 👉 **\[텍스트 분리]** 추가
* 설정:

  * **입력:** 위에서 수정된 텍스트
  * **구분자:** `줄 바꿈`

---

## ✅ 4단계: 현재 날짜 가져오기

* 🔍 검색창에 `현재 날짜`
* 👉 **\[현재 날짜 가져오기]**

---

## ✅ 5단계: 날짜 형식 지정

* 🔍 검색창에 `날짜 형식 지정`
* 👉 **\[날짜 형식 지정]**
* 설정:

  * **날짜:** 앞에서 가져온 현재 날짜
  * **형식:** `yyyy년 M월 d일`

---

## ✅ 6단계: 폴더명 사용자 입력 받기

* 🔍 검색창에 `질문 표시`
* 👉 **\[질문 표시]**
* 설정:

  * **질문:** `"메모를 저장할 iCloud 폴더명을 입력하세요"`
  * **기본 응답:** 위에서 만든 날짜 형식 (예: `2025년 6월 8일`)

> 입력 결과는 변수로 저장됨 (예: `폴더명`)

---

## ✅ 7단계: 반복 설정 (각 줄마다 메모 생성)

* 🔍 검색창에 `반복`
* 👉 **\[반복]**
* 항목: **텍스트 분리 결과**를 반복 대상으로 지정

---

### ▶️ 반복 블록 안에 다음 작업 추가:

#### ✅ 7-1단계: 새 메모 만들기

* 🔍 검색창에 `메모 만들기`
* 👉 **\[새 메모 만들기]**
* 설정:

  * **내용:** `반복 항목`
  * **폴더:** 위에서 입력받은 `폴더명`
  * **계정:** `iCloud`

* 첫 줄은 폴더명으로 사용되고

## 🔧 단축어 실제 만들기 (단계별 안내)

---

### ① 단축어 앱 실행 및 새 단축어 만들기

1. 아이폰에서 **단축어** 앱 실행
2. 하단 메뉴에서 **\[내 단축어]** 탭
3. 우측 상단 **`+` 버튼** 누르기 → 새 단축어 생성
4. 상단에 **단축어 이름** 입력 (예: `줄별 메모 생성`)
5. 중앙 하단의 **\[동작 추가]** 버튼 누르기

---

### ② 클립보드에서 텍스트 가져오기

1. 검색창에 `클립보드` 입력
2. → **\[클립보드에서 텍스트 가져오기]** 선택
3. 파란 블록으로 추가되었는지 확인

✅ 이 단계는 복사된 텍스트를 불러오는 역할이야.

---

### ③ 텍스트를 줄 단위로 나누기

1. **+ 동작 추가**
2. 검색창에 `텍스트 분리` 입력
3. → **\[텍스트 분리]** 선택
4. 설정 확인:

   * **입력값:** 위의 클립보드 텍스트
   * **구분자:** `줄 바꿈` 그대로 둬

✅ 이제 각 줄이 하나의 목록 항목이 된 상태야.

---

### ④ 첫 줄을 폴더명으로 추출

1. **+ 동작 추가**
2. 검색창에 `항목 가져오기` 입력
3. → **\[목록에서 항목 가져오기]** 선택
4. 설정:

   * **목록:** 위에서 나눈 목록
   * **항목 번호:** `1`

📌 이 단계는 "첫 줄 → 폴더명" 용도로 가져오는 거야.

---

### ⑤ 반복문 만들기 (각 줄마다 메모 생성 시도)

1. **+ 동작 추가**
2. 검색창에 `반복` 입력
3. → **\[반복]** 선택
4. **대상 목록:** ③의 "텍스트 분리 결과"

✅ 이제 각 줄을 순서대로 반복 처리하게 될 거야.

---

### ⑥ 반복 인덱스(현재 몇 번째 줄인지) 알아내기

1. 반복 블록 안으로 들어가서
2. **+ 동작 추가**
3. 검색창에 `반복 인덱스` 또는 `Repeat Index` 입력
4. → **\[반복 인덱스]** 선택

📌 이건 현재 줄이 몇 번째인지 알려주는 기능이야.

---

### ⑦ 조건문: 인덱스가 1보다 큰 경우에만 메모 생성

1. 반복 안에서 **+ 동작 추가**
2. 검색창에 `조건` 또는 `If` 입력
3. → **\[조건문 (If)]** 선택
4. 설정:

   * **조건:** `반복 인덱스` > `1`

✅ 첫 줄은 건너뛰고, 두 번째 줄부터만 처리됨.

---

### ⑧ 조건 만족 시 → 새 메모 생성

1. 조건문 안쪽(참일 때)에 **+ 동작 추가**
2. 검색창에 `메모` 입력
3. → **\[새 메모 만들기]** 선택
4. 설정:

   * **내용:** `반복 항목` (현재 줄 내용)
   * **폴더:** ④단계에서 가져온 폴더명 항목
   * **계정:** `iCloud`

✅ 이 줄이 메모로 저장될 내용이야.

3. **완료** 누르기

## 🔷 2단계: 새 메모 만들기 블록에서 폴더 자리에 이 값을 넣음

| 항목  | 값                           |
| --- | --------------------------- |

## 🔷 3단계: 메모 앱에서 자동으로 폴더가 생성됨

  → **“일정 정리”라는 폴더가 자동으로 생기고**

✅ **따로 폴더 생성하는 명령 없이**,
📌 **“폴더 자리에 없는 이름을 넣기만 하면 자동 생성”되는 게 핵심이야**

| 단계 | 동작           | 설명                                       |

# ✅ “첫 줄 가져오기” 값을 “폴더” 항목에 넣는 방법

---

## 🎯 전체 흐름 요약

1. 줄 단위로 텍스트 나눔 → `텍스트 분리`
2. 첫 줄 가져오기 → `목록에서 첫 번째 항목 가져오기`
3. 이 값을 **메모 생성 블록의 ‘폴더’ 칸**에 매직 변수로 넣기

---

## 📱 아이폰에서 따라 하기

---

### ✅ 1단계: 먼저 “첫 줄 가져오기” 블록이 있는지 확인

* 블록 이름은 보통 이렇게 생겼어:

  ```
  텍스트 분리에서 첫 번째 항목 가져오기
  ```

이 블록은 **폴더명으로 쓸 텍스트**를 저장해주는 역할이야.

---

### ✅ 2단계: “새 메모 만들기” 블록에서 **‘폴더’ 자리를 탭**

1. “폴더”라고 되어 있는 회색 박스를 톡 누르면
   아래쪽에 파란색 변수들이 쭉 뜰 거야

2. 그중에서 **“첫 번째 항목 가져오기 결과”**, 또는
   **“째 항목”**, **“항목 가져오기 결과”**, **“목록에서 항목”**
   같은 이름이 보이면 그걸 선택해줘!

✅ 이게 바로 **앞에서 꺼낸 첫 줄의 값이야**
📌 즉, `"폴더명"`이 이 값으로 설정되는 거야.

---

## 🔍 변수 이름은 상황 따라 다를 수 있어!

| 네 화면에 뜰 수 있는 이름 | 의미        | 선택해도 됨? |
| --------------- | --------- | ------- |
| `째 항목`          | 첫 줄 가져온 값 | ✅       |
| `항목 가져오기 결과`    | 동일        | ✅       |
| `목록의 항목`        | 반복 항목 (X) | ❌       |
| `"반복 인덱스"`      | 줄 번호 (X)  | ❌       |

> ❗ “반복 항목”과 혼동하지 마 — 이건 메모 내용용이고
> 📁 \*\*폴더 자리엔 반드시 “첫 줄 가져온 값”\*\*을 넣어야 해!

## 아이폰 단축어앱에서 클립보드의 내용을 한 줄씩 읽는 로직

아이폰의 단축어(Shortcuts) 앱을 이용하면 클립보드에 복사된 텍스트를 불러와 한 줄씩 처리할 수 있습니다. 아래는 이를 구현하는 기본적인 단계와 핵심 로직입니다.

---

**1. 단축어 생성 및 클립보드 불러오기**

- 단축어 앱을 실행하고, 새 단축어를 만듭니다.
- '동작 추가'를 누른 후, '클립보드 가져오기'를 선택합니다. 이 동작으로 현재 클립보드에 저장된 텍스트가 단축어 내 변수로 저장됩니다[2][4][5].

**2. 텍스트를 한 줄씩 분리하기**

- '텍스트 분리' 또는 '분할' 동작을 추가합니다.
- 분리 기준(구분자)을 '새 줄'로 설정합니다. 이렇게 하면 클립보드의 텍스트가 줄바꿈(엔터) 기준으로 분리되어 각 줄이 리스트(목록)로 저장됩니다.

**3. 각 줄을 반복 처리하기**

- '반복하기' 또는 '각 항목 반복' 동작을 추가합니다.
- 앞 단계에서 분리된 리스트(줄 목록)를 반복 대상으로 지정합니다.
- 반복문 내부에서 각 줄에 원하는 동작(예: 알림 표시, 다른 앱으로 전달 등)을 추가할 수 있습니다.

---

### 예시: 한 줄씩 알림으로 표시하는 단축어 흐름

1. **클립보드 가져오기**
2. **텍스트 분리** (구분자: 새 줄)
3. **각 항목 반복**
   - **알림 표시** (내용: 반복 중인 항목)

---

### 참고 사항

- 클립보드에는 마지막 복사한 내용만 저장됩니다. 여러 복사 기록을 관리하려면 별도의 앱이 필요합니다[4].
- '클립보드 가져오기' 동작은 단축어 앱에서 공식적으로 지원하는 기능입니다[5][6].

## ✅ 3단계: 조건문 만들기 – “빈 줄이나 공백 문자만 있을 경우 제외”

> 아래 순서에 따라 단축어 앱에서 하나씩 만들어줘.
> 단축어 편집 화면에서 "동작검색"을 눌러 각각의 블록을 추가해.

---

### 🔹① \[텍스트 바꾸기] – 공백 문자 제거

1. **“동작검색”** → `텍스트 바꾸기` 검색 → 추가

2. 설정 내용:

   * "입력" → `반복 항목`으로 설정
   * "찾을 텍스트" → **공백 문자**로 설정 (스페이스 하나 입력)
   * "바꿀 텍스트" → **비워둠** (빈 문자열로 설정)

   이 동작은 `반복 항목`에서 모든 공백을 제거한 문자열을 만들어줘.

3. 이 결과를 변수로 저장 (예: `정제된 텍스트`)

   * 블록 우측 상단에 있는 작은 설정 버튼(▶) → "결과 저장" 체크하고
     변수 이름: `정제된 텍스트`

---

### 🔹② \[조건문] – “정제된 텍스트가 비어 있지 않을 때만 통과”

1. **“동작검색”** → `조건문` 검색 → 추가
2. 조건 설정:

   * 첫 번째 조건:

     * `정제된 텍스트` → `텍스트` → `is not` → `""` (빈 문자열)
   * 두 번째 조건:

     * `반복 항목` → `텍스트에 포함되지 않음` → `"제외"` (또는 네가 제외하고 싶은 단어)
3. 위 두 조건을 **모두 만족해야 하므로**, "AND(모두 충족)" 선택

---

### 🔹③ 조건문이 \*\*참(True)\*\*일 경우에만 \[메모 생성]

조건이 만족될 때(`조건이 참일 때`)만
