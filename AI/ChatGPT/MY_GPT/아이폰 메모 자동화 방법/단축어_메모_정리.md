
# 📘 단축어 프로젝트 요약 (iOS 18.5 기준)

---

## ✅ 1. 단축어 기능 목표

- 📋 클립보드 내용을 줄 단위로 나누고
- 📁 첫 줄은 **폴더명**으로 사용
- 📄 두 번째 줄부터는 **iCloud 메모 앱에 개별 메모로 생성**
- ❌ 빈 줄 또는 특정 단어가 포함된 줄은 **메모에서 제외**
- 🔁 위 과정을 자동화한 단축어 만들기

---

## ✅ 2. 사용한 주요 단축어 블록 흐름 (기능 순서)

1. **클립보드 가져오기**  
   → 복사된 텍스트 불러옴

2. **텍스트 분리 (줄바꿈 기준)**  
   → 줄마다 목록으로 나눔

3. **첫 줄 가져오기**  
   → 폴더명으로 사용될 텍스트 저장

4. **반복문: 목록 전체 반복**  
   → 한 줄씩 처리

5. **반복 인덱스 가져오기**  
   → 몇 번째 줄인지 파악

6. **조건문 1: 인덱스 > 1**  
   → 첫 줄(폴더명)은 메모로 만들지 않음

7. **조건문 2: 유효한 메모인지 확인**  
   → 빈 줄 또는 제외 단어 포함 시 생략

   - `반복 항목 ≠ ""`
   - `반복 항목에 "제외" 포함되지 않음`
   - 이 두 조건을 `AND`로 연결

8. **메모 생성**  
   - 텍스트: `반복 항목`  
   - 폴더: `첫 줄에서 가져온 값`  
   - 계정: `iCloud`

9. **단축어 종료**  
   - 메시지 표시: “메모 생성 완료”

---

## ✅ 3. 구현 시 참고 사항

| 주제 | 설명 |
|------|------|
| 🔠 "동작 추가" → iOS 18.5에서는 "동작검색"으로 표시됨 |
| 📁 폴더는 iCloud 기준이며, **자동 생성됨** |
| 📂 하위 폴더 개념은 메모 앱에서 지원하지 않음 (단일 계층 구조) |
| 🛑 단축어 강제 종료는 앱 스와이프 종료 또는 “단축어 종료하기” 사용 |
| 📋 메모 생성 후 앱이 종료되지 않는 문제는 종료 블록 추가로 해결됨 |

---

## ✅ 4. 다음 작업을 위해 기억할 것

- 📎 단축어 실행 흐름은 "반복 인덱스" + "조건문 두 개" 조합이 핵심
- 🧹 메모로 만들지 않을 줄은 **조건문 2에서 걸러내야 함**
- 💬 단축어 종료 메시지는 직접 작성 가능

---

## ✅ 향후 이어서 작업할 수 있는 항목 예시

- 📁 폴더명 자동 생성 규칙 개선 (날짜+제목 등)
- 📝 메모 제목 자동 설정 (예: “할 일 1”, “할 일 2”)
- 🔧 텍스트에서 불필요한 기호 제거
- 📎 단축어 파일(.shortcut) 공유 버전 생성
- 📸 실제 사용화면 단계별 이미지로 구성한 가이드 제작
