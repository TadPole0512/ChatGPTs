# 📱 아이폰 단축어: 줄 단위 메모 생성 + 사용자 수정 및 iCloud 폴더 관리

## 🎯 목적
복사한 텍스트를 줄마다 나누어 iCloud 메모에 각각 다른 메모로 저장하되,  
**사용자가 직접 폴더명을 정하고, 텍스트를 확인/수정할 수 있는 기능까지 포함**한 단축어를 만든다.

---

## ✅ 전체 기능 요약

- 클립보드 텍스트 불러오기
- 줄바꿈 기준으로 텍스트 분리
- 오늘 날짜(`2025년 6월 8일`)를 기본값으로 사용자에게 폴더명 입력받기
- 입력한 이름으로 iCloud 메모 앱 내 폴더에 저장
- 클립보드 내용을 사용자에게 보여주고 편집 기회 제공
- 사용자가 편집한 최종 텍스트를 줄바꿈 기준으로 메모 생성

---

## 🧩 단축어 단계별 구성

### 1. 클립보드에서 텍스트 가져오기
- 작업: **클립보드에서 텍스트 가져오기**

### 2. 클립보드 내용 사용자에게 보여주기 (수정 가능)
- 작업: **텍스트 편집**
  - 입력: 클립보드 텍스트
  - 설명: "메모로 만들기 전 텍스트를 수정하세요 (줄마다 하나씩 메모 생성됨)"

### 3. 편집된 텍스트 줄바꿈 기준으로 분리
- 작업: **텍스트 분리**
  - 구분자: 줄바꿈 (`\n`)

### 4. 오늘 날짜 구하기
- 작업: **현재 날짜 가져오기**
- 작업: **날짜 형식 지정**
  - 형식: `yyyy년 M월 d일`

### 5. 사용자에게 폴더명 입력 요청
- 작업: **질문 표시**
  - 질문: "저장할 메모 폴더 이름을 입력하세요"
  - 기본 응답: `2025년 6월 8일`

### 6. 반복 (분리된 각 줄마다)
- 작업: **반복**
  - 항목: 텍스트 분리 결과

#### 반복 내부 작업
- 작업: **새 메모 만들기**
  - 내용: 반복 항목
  - 폴더: 사용자 입력 폴더명
  - 계정: **iCloud**

## 💡 추가 요청 기능 점검

| 기능 | 가능 여부 | 설명 |
|------|------------|------|
| 📁 iCloud에 메모 저장 | ✅ 가능 | 단축어에서 메모 계정 지정 가능 |
| 🆕 메모 생성 전 폴더 생성 | ⚠️ 조건부 가능 | 폴더 이름 지정하면 iCloud에서 자동 생성됨 (직접 생성 명령은 불가) |
| 📅 날짜 예시로 폴더명 제시 | ✅ 가능 | `현재 날짜` → `질문 기본 응답`으로 연결 가능 |
| 👀 클립보드 내용 확인 및 수정 | ✅ 가능 | "텍스트 편집" 항목 활용 |
| 📝 수정된 텍스트로 메모 생성 | ✅ 가능 | 수정 후 분리해서 사용 가능 |
| 📂 폴더 선택 시 팝업으로 해당 메모 제목 보기 | ❌ 불가능 | iOS 단축어에선 메모 리스트 읽기 불가능 |
| 🧾 기존 메모 내용 수정 | ❌ 불가능 | 단축어에선 기존 메모 편집 API 없음 |

---

## 🚫 제한 사항
- 폴더 안 메모 제목 목록을 보여주는 기능은 현재 iOS 단축어에서 지원하지 않음
- 기존 메모 수정도 단축어에서는 제한됨 (읽기 전용 불가)

---

## 📝 예시 흐름

```
클립보드 내용 복사 →
사용자가 확인 및 수정 →
줄마다 분리 →
오늘 날짜 형식 폴더명 입력 →
입력한 폴더 안에 메모 생성 (iCloud)
```
