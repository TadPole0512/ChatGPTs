## 🔰 1. 기본 개념 - NotebookLM이 뭐예요?

### 🎯 한줄 요약
> NotebookLM은 "내 자료만 보고 답하는 똑똑한 AI 노트"예요!

### 📖 쉬운 비유로 이해하기

```
🏫 학교 교실 상황으로 비교하면:

일반 ChatGPT = 모든 걸 다 아는 백과사전 선생님
             (세상 모든 지식을 알지만, 내 자료는 모름)

NotebookLM = 내가 준 교과서만 보는 과외 선생님
           (내 자료만 읽고, 그걸로만 답해줌)
```

### 💡 주요 특징

| 특징 | 설명 | 장점 |
|------|------|------|
| 📁 소스 기반 | 내가 올린 자료만 참고 | 잘못된 정보(환각) 줄어듦 |
| 📌 인용 표시 | 어느 페이지에서 가져왔는지 알려줌 | 신뢰도 높음, 과제에 쓰기 좋음 |
| 🎧 오디오 변환 | 글을 팟캐스트처럼 만들어줌 | 운동하면서 들을 수 있음 |
| 👥 협업 가능 | 여러 명이 함께 사용 | 조별 과제에 최적 |

### ⚠️ 주의사항

> **[확인됨]** 다음 내용은 반드시 알아두세요!

- ❌ **저작권**: 불법 복제 자료 업로드 금지
- ❌ **개인정보**: 주민번호, 비밀번호 같은 민감 정보 올리지 마세요
- ❌ **100% 신뢰 금지**: AI도 가끔 실수해요. 중요한 내용은 꼭 원본 확인!
- ✅ **언어**: 한국어, 영어 등 다양한 언어 지원

## 🔰 2. 시작하기 - 계정 만들고 접속하기

### 🎯 한줄 요약
> 구글 계정만 있으면 무료로 바로 시작!

### 🛠 준비물 체크리스트

- [ ] 구글 계정 (Gmail)
- [ ] 인터넷 연결된 컴퓨터 또는 스마트폰
- [ ] 크롬(Chrome) 브라우저 권장 (다른 브라우저도 가능)

### 🔄 따라하기 (클릭 경로)

```
1단계: notebooklm.google.com 접속
   ▼
2단계: [Get started] 또는 [시작하기] 버튼 클릭
   ▼
3단계: 구글 계정으로 로그인
   ▼
4단계: 약관 동의 (개인정보 처리방침 읽어보기)
   ▼
5단계: 첫 화면(홈) 도착! 🎉
```

### 👀 첫 화면 구성 미리보기

```
┌─────────────────────────────────────┐
│ 🔍 [검색창]         [설정⚙️] [프로필👤] │
├─────────────────────────────────────┤
│                                     │
│  📝 내 노트북 목록                    │
│  ┌─────┐ ┌─────┐ ┌─────┐          │
│  │새로운│ │과학 │ │수학 │          │
│  │노트북│ │수업 │ │공부 │          │
│  └─────┘ └─────┘ └─────┘          │
│                                     │
│  [+ New Notebook] 버튼 ← 여기 클릭!  │
│                                     │
└─────────────────────────────────────┘
```

### ✅ 완료 점검

- [ ] notebooklm.google.com에 정상 접속됨
- [ ] 내 구글 계정 이름이 오른쪽 위에 보임
- [ ] "New Notebook" 버튼이 보임
- [ ] 왼쪽 메뉴에서 내 노트북 목록 확인 가능

### ⚠️ 흔한 실수 & 해결법

| 문제 | 원인 | 해결 방법 |
|------|------|----------|
| 로그인 안 됨 | 구글 계정 없음 | Gmail 먼저 만들기 |
| 화면이 안 보임 | 브라우저 문제 | 크롬으로 다시 시도 |
| 영어로만 나옴 | 언어 설정 | 설정⚙️ ▶ Language ▶ 한국어 선택 |

### 💡 응용 팁

1. **북마크 등록**: 자주 쓴다면 주소를 즐겨찾기에 추가하세요
2. **앱처럼 사용**: 크롬에서 "설정 ▶ 앱 설치" 하면 바탕화면 아이콘 생성!
3. **모바일**: 스마트폰 브라우저에서도 동일하게 작동 (앱 없음, 웹으로만 사용)

## 🔰 3. 노트북 만들기 - 새 프로젝트 시작

### 🎯 한줄 요약
> 노트북 = 하나의 주제를 담는 큰 폴더 (예: "과학 공부", "여행 계획")

### 🛠 준비물

- [ ] NotebookLM 로그인 상태
- [ ] 어떤 주제로 만들지 생각해두기 (예: 수학 공부, 독서 노트)

### 🔄 따라하기

```
홈 화면
   ▼
[+ New Notebook] 클릭
   ▼
노트북 이름 입력창 나타남
   ▼
"과학 실험 정리" 같은 이름 입력
   ▼
[Create] 또는 [만들기] 클릭
   ▼
✨ 새 노트북 화면 열림!
```

### 👀 새 노트북 화면 구성

```
┌──────────────────────────────────────────┐
│ ← [뒤로] "과학 실험 정리" 📝 [공유] [설정]  │
├──────────────────────────────────────────┤
│ 왼쪽 패널 (Sources)   │  오른쪽 패널 (Chat) │
│                      │                    │
│ 📄 Sources (자료)     │  💬 대화창          │
│                      │                    │
│ [+ Add sources]      │  "자료에 대해       │
│                      │   질문하세요"       │
│ (아직 비어있음)        │                    │
│                      │  [입력창]           │
└──────────────────────┴────────────────────┘
```

### ✅ 완료 점검

- [ ] 노트북 이름이 상단에 표시됨
- [ ] 왼쪽에 "Sources" 패널 보임
- [ ] 오른쪽에 대화창 보임
- [ ] "+ Add sources" 버튼 확인

### ⚠️ 흔한 실수

| 실수 | 결과 | 올바른 방법 |
|------|------|------------|
| 이름 안 지음 | "Untitled" 노트북 생성됨 | 나중에 수정 가능 (상단 이름 클릭) |
| 주제가 너무 넓음 | 자료가 섞여서 헷갈림 | 주제별로 노트북 분리 (예: "중간고사용", "기말고사용") |
| 삭제 방법 모름 | 노트북이 쌓임 | 홈에서 노트북 우클릭 ▶ Delete |

### 💡 응용 팁

**노트북 정리 예시:**
```
📁 학교
  ├ 📝 수학 - 1학기
  ├ 📝 과학 - 생물 단원
  └ 📝 영어 - 문법 정리

📁 개인
  ├ 📝 독서 노트 - 2024
  └ 📝 여행 계획 - 제주도
```

## 🔰 4. 자료 추가하기 - 내 자료를 AI에게 보여주기

### 🎯 한줄 요약
> 문서, 웹페이지, 구글 드라이브 파일을 드래그만 하면 끝!

### 🛠 지원하는 자료 형식

| 종류 | 형식 | 최대 크기/개수 |
|------|------|---------------|
| 📄 문서 | PDF, TXT, MD | 최대 50개 소스 |
| 🌐 웹페이지 | URL 주소 | 제한 없음 |
| 📁 Google Drive | Docs, Sheets, Slides | 최대 50개 |
| 📋 복사-붙여넣기 | 텍스트 직접 입력 | 제한 없음 |

> ⚠️ **[확인됨]** 워드(.docx)는 직접 지원 안 함 → PDF로 변환 후 업로드!

### 🔄 따라하기 - 방법 1: 파일 업로드

```
노트북 화면 왼쪽 패널
   ▼
[+ Add sources] 클릭
   ▼
팝업창에서 [Upload] 탭 선택
   ▼
[Select files] 클릭 또는 파일 드래그
   ▼
파일 선택 (여러 개 한번에 가능)
   ▼
⏳ 업로드 중... (진행바 표시)
   ▼
✅ 왼쪽에 파일 이름 나타남!
```

### 🔄 따라하기 - 방법 2: 웹페이지 추가

```
[+ Add sources] 클릭
   ▼
[Website URL] 탭 선택
   ▼
주소창에 URL 붙여넣기
  (예: https://ko.wikipedia.org/wiki/광합성)
   ▼
[Add] 버튼 클릭
   ▼
✅ 웹페이지 내용이 자동으로 저장됨!
```

### 🔄 따라하기 - 방법 3: 구글 드라이브 연결

```
[+ Add sources] 클릭
   ▼
[Google Drive] 탭 선택
   ▼
내 드라이브 파일 목록 나타남
   ▼
원하는 파일 체크
   ▼
[Add selected] 클릭
   ▼
✅ 드라이브 파일 추가 완료!
```

### 🔄 따라하기 - 방법 4: 텍스트 직접 입력

```
[+ Add sources] 클릭
   ▼
[Paste text] 탭 선택
   ▼
제목 입력 (예: "선생님 필기 내용")
   ▼
큰 입력창에 텍스트 붙여넣기
   ▼
[Add] 클릭
   ▼
✅ 텍스트가 하나의 소스로 저장됨!
```

### 👀 추가된 소스 확인하기

```
왼쪽 패널 (Sources)
┌─────────────────────┐
│ 📄 과학교과서.pdf    │ ← 파일명 클릭하면 내용 미리보기
│    45 pages         │
│                     │
│ 🌐 광합성 - 위키    │
│    Web source       │
│                     │
│ 📁 실험보고서.docs   │
│    From Drive       │
└─────────────────────┘
```

### ✅ 완료 점검

- [ ] 왼쪽 패널에 내가 추가한 자료가 보임
- [ ] 각 자료에 아이콘(📄🌐📁)이 표시됨
- [ ] 자료 이름 클릭하면 내용 미리보기 가능
- [ ] 오른쪽 대화창에 "N sources added" 메시지 확인

### ⚠️ 흔한 실수 & 해결법

| 문제 | 원인 | 해결 방법 |
|------|------|----------|
| PDF가 안 올라감 | 파일이 너무 큼 (100MB+) | 파일 압축 또는 나눠서 업로드 |
| 웹페이지가 빈 화면 | 로그인 필요한 사이트 | 공개된 페이지만 가능 |
| 한글이 깨짐 | 인코딩 문제 | UTF-8로 저장 후 재업로드 |
| 업로드 버튼 없음 | 50개 초과 | 기존 소스 삭제 후 추가 |

### 💡 응용 팁

**프롬프트 예시 1: 소스 정리**
```
"내 소스들을 주제별로 분류해줘"
→ AI가 자동으로 카테고리 제안
```

**프롬프트 예시 2: 중복 확인**
```
"같은 내용이 중복된 소스가 있어?"
→ 비슷한 내용 찾아줌
```

**실전 팁:**
- 📌 **태그 활용**: 소스 이름에 [중요], [시험], [참고] 같은 라벨 붙이기
- 🗂 **순서 변경**: 드래그로 소스 순서 바꾸기 (최신 자료 위로)
- 🗑 **삭제**: 소스 우클릭 ▶ Remove source

## 🔥 5. 질문하고 답변받기 - AI에게 물어보기

### 🎯 한줄 요약
> 내 자료만 보고 답하는 맞춤형 AI 선생님!

### 🛠 준비물

- [ ] 노트북에 최소 1개 이상의 소스 추가됨
- [ ] 질문하고 싶은 내용 생각해두기

### 🔄 따라하기 - 기본 질문

```
오른쪽 대화창 하단 입력창
   ▼
질문 입력 (예: "광합성이 뭐야?")
   ▼
[Send] 또는 Enter 키
   ▼
⏳ AI가 내 소스를 읽는 중...
   ▼
✨ 답변 + 인용 출처 표시!
```

### 👀 답변 화면 구성

```
┌────────────────────────────────────┐
│ 💬 내 질문: "광합성이 뭐야?"         │
├────────────────────────────────────┤
│ 🤖 AI 답변:                         │
│                                    │
│ 광합성은 식물이 빛 에너지를 이용해    │
│ 이산화탄소와 물로 포도당을 만드는     │
│ 과정입니다. [1]                      │
│                                    │
│ 📌 인용 출처:                        │
│ [1] 과학교과서.pdf, p.23            │
│     "...빛 에너지를 화학 에너지로..." │
│     [원문 보기 →]                    │
└────────────────────────────────────┘
```

### ✅ 완료 점검

- [ ] 질문에 대한 답변이 나타남
- [ ] 답변에 [1], [2] 같은 인용 번호 표시
- [ ] 인용 번호 클릭하면 원문 위치로 이동
- [ ] 답변 아래 "인용 출처" 섹션 확인

### 🎯 질문 유형별 활용법

| 질문 유형 | 예시 | 언제 유용한가 |
|---------|------|-------------|
| 📖 개념 설명 | "삼투압이 뭐야?" | 기본 개념 이해할 때 |
| 📊 요약 | "1장 내용을 3줄로 요약해줘" | 빠르게 훑어볼 때 |
| 🔍 비교 | "A와 B의 차이점은?" | 헷갈리는 개념 정리 |
| 💡 예시 | "광합성 예시 3개 들어줘" | 구체적 사례 필요할 때 |
| ✅ 확인 | "내 이해가 맞아? [설명]" | 자가 점검할 때 |

### 🔄 고급 질문 패턴

**패턴 1: 소스 지정 질문**
```
❌ 나쁜 예: "이게 맞아?"
✅ 좋은 예: "과학교과서.pdf에서 광합성 과정 단계를 설명해줘"
```

**패턴 2: 형식 지정 질문**
```
"광합성 과정을 표로 만들어줘"
"1단계, 2단계, 3단계로 나눠서 설명해줘"
"초등학생도 이해할 수 있게 쉽게 설명해줘"
```

**패턴 3: 범위 지정 질문**
```
"1~3장에서 중요한 개념 5개만 뽑아줘"
"결론 부분만 요약해줘"
"서론에서 저자의 주장이 뭐야?"
```

### ⚠️ 흔한 실수 & 해결법

| 문제 | 원인 | 해결 방법 |
|------|------|----------|
| "소스에서 찾을 수 없음" | 자료에 그 내용 없음 | 질문 바꾸거나 자료 추가 |
| 답변이 너무 짧음 | 질문이 모호함 | 구체적으로 다시 물어보기 |
| 엉뚱한 답변 | 자료가 너무 많음 | 소스 범위 좁혀서 질문 |
| 인용이 안 보임 | AI가 확신 없음 | 같은 질문 다시 해보기 |

### 💡 응용 팁 - 프롬프트 예시

**레벨 1: 기초 이해**
```

```

**레벨 3: 시험 대비**
```

### Q1. 무료인가요? 유료인가요?

**A.** 현재 무료입니다! (2025년 1월 기준)

### Q2. ChatGPT랑 뭐가 달라요?

**A.** 핵심 차이:

| | ChatGPT | NotebookLM |
|---|---------|------------|
| 지식 범위 | 전 세계 모든 지식 | 내가 준 자료만 |
| 환각 위험 | 높음 | 낮음 (출처 표시) |
| 인용 | 부정확 | 정확 (페이지/문단) |
| 최적 용도 | 일반 질문 | 학습/리서치 |

### Q3. 어떤 파일 형식을 지원하나요?

**A.** 지원 형식:
- ✅ PDF
- ✅ TXT
- ✅ Markdown (.md)
- ✅ Google Docs/Sheets/Slides
- ✅ 웹페이지 (URL)
- ❌ 워드(.docx) → PDF 변환 필요
- ❌ 파워포인트(.pptx) → PDF 변환 필요

### Q4. 한국어 지원되나요?

**A.** ✅ 네! 완벽 지원
- 한국어 문서 업로드 OK
- 한국어로 질문 OK
- 한국어 답변 OK
- 오디오도 한국어로 생성 가능

**A.** ❌ 안 됩니다
- 인터넷 필수
- 클라우드 기반 서비스

### Q7. 내 데이터는 안전한가요?

**A.** Google 보안 정책 적용
- 구글 드라이브 수준 보안

```

### 시나리오 B: 회의 준비 (직장인용)

**상황**: 프로젝트 킥오프 미팅 준비

```
1단계: 노트북 만들기
→ 이름: "신제품 출시 프로젝트"

2단계: 자료 수집
→ 기획서 PDF
→ 경쟁사 분석 웹페이지
→ 이전 프로젝트 회의록 (구글 독스)

3단계: 분석
→ "주요 의사결정 포인트 5개 정리해줘"
→ "경쟁사 대비 우리 강점/약점 분석해줘"
→ "예상 리스크와 대응 방안을 표로 만들어줘"

4단계: 발표 준비
→ "5분 발표용 요점 정리해줘"
→ "예상 질문 10개와 답변 만들어줘"
→ 오디오 브리핑으로 리허설

5단계: 공유
→ 팀원들에게 Editor 권한으로 공유
→ 각자 의견을 댓글로 추가
→ 최종 노트 공동 작성
```

### 시나리오 C: 논문 비교 (대학원생용)

**상황**: 문헌 고찰 (Literature Review)

```
1단계: 노트북 만들기
→ 이름: "기계학습 논문 리뷰 2024"

2단계: 논문 업로드
→ 관련 논문 PDF 10개 추가
→ 핵심 논문은 [중요] 태그

3단계: 비교 분석
→ "10개 논문의 연구 방법을 비교 표로 만들어줘"
→ "공통적으로 인용되는 핵심 이론 정리해줘"
→ "시간순으로 연구 트렌드 변화 분석해줘"

4단계: 인사이트 도출
→ "아직 연구 안 된 gap이 뭐야?"
→ "각 논문의 한계점 정리해줘"
→ "내 연구 주제로 적합한 방법론 추천해줘"

5단계: 문헌 고찰 작성
→ "이 내용으로 문헌 고찰 초안 작성 (3000단어)"
→ 인용 포함해서 정리
→ 교수님과 공유 (Commenter 권한)
```
