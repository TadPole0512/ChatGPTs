### 📜 **핵심 지침 (Core Instructions)**

**1. 🎯 목표 (Goal):**
- **대상:** NotebookLM을 처음 사용하는 완전 초보자
- **내용:** 시작부터 고급 활용까지 NotebookLM의 모든 기능 포함
- **수준:** 복잡한 개념은 비유와 예시를 통해 초등학생도 이해할 수 있도록 설명
- **구성:** 사용자의 자연스러운 작업 흐름(시작하기 → 핵심 기능 → 고급 활용 → 실전 예제)에 따라 체계적으로 구성

**2. ✨ 시각적 & 형식적 요구사항 (Visual & Formatting Requirements):**
- **GUI 스타일 우선:** 모든 설명은 카드, 체크리스트, 접기/펼치기(Accordion) 등 그래픽 사용자 인터페이스(GUI) 스타일을 최우선으로 적용하여 가독성을 극대화해 주세요.
- **아이콘과 이모지 활용:** 각 섹션과 기능 설명 앞에 역할에 맞는 아이콘(🧭, ✨, 🚀, 💡 등)을 붙여 시각적 구분을 명확하게 해주세요.
- **카드 형식 요약:** 각 기능 설명 시작 부분에 핵심 내용을 요약한 `카드 블록`을 배치해 주세요.
- **단계별 가이드:** 모든 기능 사용법은 `체크리스트` 형식의 단계별(Step-by-step) 가이드로 제시하여 사용자가 하나씩 따라 할 수 있게 만들어주세요.
- **콜아웃 박스 활용:** `> [!TIP]` (꿀팁), `> [!NOTE]` (참고), `> [!WARNING]` (주의) 같은 콜아웃 박스를 적극적으로 사용하여 정보를 시각적으로 강조해 주세요.
- **접기/펼치기 기능:** 상세 설명이나 심화 내용은 `<details>` 태그를 사용한 접기/펼치기(Collapsible Section) 안에 넣어 화면을 깔끔하게 유지해 주세요. 제목은 "🔍 자세히 알아보기" 와 같이 직관적으로 작성해 주세요.
- **시각 자료 요청:** 설명에 이미지가 필수적인 부분에는 ``과 같이 어떤 이미지가 들어가야 할지 구체적인 캡션을 포함한 이미지 태그를 삽입해 주세요.

**3. 🧠 고급 활용법 생성 (Advanced Application Generation):**
- 가이드의 '고급 활용편'에서는 제공된 **'Genius Insight Formula'**와 **'Creative Connection Matrix'** 사고 모델을 적용해야 합니다.
- 이 두 가지 공식을 활용하여, 사용자가 NotebookLM을 단순히 요약 도구가 아닌, 창의적 문제 해결 및 아이디어 발상 도구로 사용할 수 있는 **'천재적인 활용 아이디어' 10가지 이상**을 구체적인 시나리오와 함께 제시해 주세요. (예: 시장 분석 보고서 초안 작성, 소설 시놉시스 개발, 과학 논문 데이터 분석 등)
- 각 아이디어는 어떤 사고 공식을 어떻게 적용했는지 간단히 언급해 주세요.

**4. 📤 출력 및 전달 방식 (Output & Delivery Method):**
- **기본 출력:** 화면 출력을 우선으로 하며, 하나의 완성된 Markdown 파일 형식으로 모든 내용을 생성해 주세요.
- **파일 제공 옵션:** 만약 내용이 너무 길어질 경우, 먼저 전체 구조를 보여주는 '목차'를 제시한 후, 본문 생성을 시작해 주세요. 그리고 "요청 시 전체 내용을 .md 파일로 제공할 수 있습니다."라고 안내 문구를 추가해 주세요.
