## 🔰 1. 첫걸음 떼기: AI 놀이터 준비하기

### 1-1. NotebookLM, 그게 뭔가요?

🎯 **한줄 요약**

> 내가 올린 자료(PDF, 웹사이트, 문서 등)만 콕 집어서 공부하고 대답해주는 똑똑한 나만의 AI 비서예요.

🏠 **쉬운 비유**

> **NotebookLM**은 **'나만의 똑똑한 디지털 폴더'** 같아요.
>
> 1.  \*\*'폴더(노트북)'\*\*를 하나 만들고,
> 2.  여기에 \*\*'공부할 책이나 자료(소스)'\*\*를 넣어주면,
> 3.  \*\*'AI 비서'\*\*가 그 자료들만 달달 외워서 **질문에 척척 대답**해주고, **요약**도 해주고, **퀴즈**도 내주는 거죠\!

<br>

### 1-2. 새 노트북(폴더) 만들기

\<div style="border: 1px solid \#ddd; padding: 20px; border-radius: 10px; margin-bottom: 20px;"\>

🎯 **한줄 요약**: 앞으로 작업할 주제별로 깨끗한 새 폴더를 만드는 과정이에요.

-----

🛠 **준비물 체크리스트**

  - [ ] 구글(Google) 계정
  - [ ] 인터넷이 연결된 컴퓨터

-----

🔄 **따라하기**

1.  **NotebookLM 홈페이지 접속** ▶ 화면 오른쪽 위 **'Launch NotebookLM'** 클릭 ▶ 구글 계정으로 **로그인**
2.  화면 왼쪽 위 **'+ New notebook'** 버튼 클릭

-----

👀 **화면 미리보기 설명**

> > 왼쪽 위에 보이는 파란색 **'+ New notebook'** 버튼이 보일 거예요. 이걸 누르면 새 작업 공간이 짠\! 하고 나타나요. 앞으로 만들 노트북들은 이 버튼 아래에 차곡차곡 쌓이게 됩니다.

-----

✅ **완료 점검**

  - [ ] 화면에 'Untitled notebook'이라는 이름의 새 노트북이 생겼나요?
  - [ ] 노트북 이름을 내가 원하는 주제(예: '과학 숙제')로 바꿀 수 있나요?

-----

⚠️ **흔한 실수 & 해결법**

  - **실수**: 노트북을 너무 많이 만들어 뭐가 뭔지 헷갈려요.
  - **해결**: 노트북 이름에 날짜나 주제를 명확하게 적어주세요 (예: `241006_과학_화산폭발_리포트`)

-----

💡 **응용/고급 팁**

  - **팁 1 (수업용)**: 과목별로 노트북을 만드세요. ('국어 노트', '수학 노트', '과학 노트')
  - **팁 2 (업무용)**: 프로젝트별로 노트북을 만드세요. ('10월 마케팅 회의', '신제품 기획안')

### 1-3. 소스(자료) 추가하기

\<div style="border: 1px solid \#ddd; padding: 20px; border-radius: 10px; margin-bottom: 20px;"\>

🎯 **한줄 요약**: 똑똑한 폴더에 AI가 공부할 책과 자료를 넣어주는 단계예요.

-----

🛠 **준비물 체크리스트**

  - [ ] 컴퓨터에 저장된 PDF, 텍스트 파일
  - [ ] 구글 드라이브에 있는 문서, 프레젠테이션
  - [ ] 내용을 참고하고 싶은 웹사이트 주소

-----

🔄 **따라하기**

1.  새 노트북 화면 중앙의 **'Add sources'** 클릭
2.  원하는 자료 종류 선택: **'Google Drive'**, **'PDF'**, **'Text file'**, **'Website'** 등
3.  파일을 **선택**하거나 웹사이트 주소를 **붙여넣기** 하면 끝\!

-----

👀 **화면 미리보기 설명**

> > `Add sources`를 누르면 여러 개의 버튼이 나타나요. 컴퓨터에서 파일을 올릴 땐 `PDF`나 `Text file`을, 구글 드라이브 자료는 `Google Drive`를, 인터넷 글을 가져올 땐 `Website`를 누르면 돼요. 최대 50개, 50만 단어까지 추가할 수 있어요\!

-----

✅ **완료 점검**

  - [ ] 노트북의 왼쪽 'Sources' 패널에 내가 추가한 자료 목록이 보이나요?
  - [ ] 자료를 클릭하면 오른쪽에 내용이 잘 보이나요?

-----

⚠️ **흔한 실수 & 해결법**

  - **실수**: 복사-붙여넣기(Copied text)로 글을 넣었더니 출처 표시가 안 돼요.
  - **해결**: 긴 글은 `.txt` 파일로 저장해서 올리거나, 웹사이트 주소를 직접 넣는 것이 나중에 인용/출처 찾기에 훨씬 좋아요.

-----

💡 **응용/고급 팁**

  - **팁 1 (리서치)**: 뉴스 기사 5개의 웹사이트 주소를 한 번에 추가해서 특정 사건에 대한 여러 관점을 비교해보세요.
  - **팁 2 (독서)**: 전자책(PDF)을 통째로 넣고, 등장인물이나 특정 사건에 대해 질문하며 깊이 있게 읽어보세요.

\</div\>

## 🔥 2. 핵심 기능 익히기: AI 비서 부려먹기

### 2-1. 질문하고 답 받기 (Q\&A)

\<div style="border: 1px solid \#ddd; padding: 20px; border-radius: 10px; margin-bottom: 20px;"\>

🎯 **한줄 요약**: 내가 올린 자료 범위 안에서만\! AI가 질문에 대답해주고, 출처까지 딱딱 알려줘요.

-----

🔄 **따라하기**

1.  자료(소스) 추가가 완료된 노트북 열기
2.  화면 맨 아래 **대화창**에 궁금한 것을 입력하고 **보내기(화살표 아이콘)** 클릭

-----

👀 **화면 미리보기 설명**

> > 대화창에 질문을 입력하면 AI가 답변을 해줘요. 신기한 건, 답변 내용 옆에 파란색 작은 숫자 `[1]`, `[2]`가 붙어있다는 점이에요. 이게 바로 **'인용(Citation)'** 표시인데, 이 숫자를 누르면 AI가 어떤 자료의 몇 페이지를 보고 대답했는지 정확하게 보여줘요\! 더 이상 가짜 정보에 속을 일이 없겠죠?

-----

✅ **완료 점검**

  - [ ] 내 질문에 대한 답변이 잘 나왔나요?
  - [ ] 답변 옆에 붙은 인용 `[숫자]` 표시를 눌렀을 때, 해당 소스 위치로 이동하나요?
  - [ ] "이 자료에 없는 내용에 대해 알려줘"라고 물었을 때, 모른다고 솔직하게 대답하나요?

-----

⚠️ **흔한 실수 & 해결법**

  - **실수**: 질문을 너무 간단하게 했더니, 원하던 답변이 안 나와요.
  - **해결**: "요약해줘" 보다는 "**초등학생도 이해하게 이 글의 핵심 주장 3가지를 요약해줘**" 처럼 구체적으로 질문해보세요.

-----

💡 **응용/고급 팁 (프롬프트 예시)**

  - **학습 프롬프트**: `업로드한 과학 교과서 3단원 내용을 바탕으로 '화산 활동의 긍정적, 부정적 영향'을 표로 정리해줘.`
  - **업무 프롬프트**: `첨부된 2개의 회의록을 비교해서 'A 프로젝트'에 대해 두 회의에서 공통으로 나온 의견과 서로 달랐던 의견을 찾아줘.`

\</div\>

### 2-2. 노트, 요약, 하이라이트

\<div style="border: 1px solid \#ddd; padding: 20px; border-radius: 10px; margin-bottom: 20px;"\>

🎯 **한줄 요약**: AI와의 대화나 소스의 중요 내용을 '노트'에 저장하고, 나중에 한 번에 모아볼 수 있어요.

-----

🔄 **따라하기**

1.  소스 문서를 읽다가 중요한 부분 **드래그하여 선택** ▶ 나타나는 메뉴에서 **'Add to note'** 클릭
2.  AI의 답변 중 마음에 드는 부분 오른쪽 위 **핀(📌) 모양 아이콘** 클릭하여 저장

-----

👀 **화면 미리보기 설명**

> > AI 답변이나 원본 소스에서 중요한 부분을 발견하면 바로바로 노트에 저장할 수 있어요. 저장된 내용들은 화면 오른쪽 'Notes' 패널에 차곡차곡 쌓여요. 나중에 이 노트들만 모아서 **'모든 노트를 하나의 문서로 합치기(Combine all notes into one doc)'** 기능으로 완벽한 요약본을 만들 수 있답니다.

-----

✅ **완료 점검**

  - [ ] 핀(📌)으로 고정한 답변이 오른쪽 노트 패널에 추가되었나요?
  - [ ] 소스에서 드래그한 문장이 노트에 잘 저장되었나요?
  - [ ] 여러 개의 노트를 합쳐서 하나의 구글 문서로 내보낼 수 있나요?

-----

💡 **응용/고급 팁**

  - **팁 1 (책 요약)**: 책 한 권(PDF)을 읽으면서 각 챕터의 핵심 문장을 노트에 추가하고, 마지막에 모든 노트를 합쳐 '나만의 책 요약본'을 만들어보세요.
  - **팁 2 (아이디어 수집)**: 여러 웹사이트를 보다가 번뜩이는 아이디어가 담긴 문장들을 노트에 모아두고, 나중에 기획서 초안을 작성할 때 활용해보세요.

\</div\>

### 2-3. 오디오 브리프 (Audio Brief)

\<div style="border: 1px solid \#ddd; padding: 20px; border-radius: 10px; margin-bottom: 20px;"\>

🎯 **한줄 요약**: AI와의 대화 내용을 바탕으로, 사람이 말하는 것처럼 자연스러운 '오디오 요약'을 만들어줘요.

-----

🔄 **따라하기**

1.  AI와 몇 차례 질문과 답변을 주고 받으며 대화 내용 쌓기
2.  대화창 위쪽에 나타나는 **헤드폰(🎧) 모양의 'Create audio brief'** 버튼 클릭
3.  AI가 자동으로 대화 내용을 요약해서 스크립트를 만들고, 오디오를 생성해줘요.

-----

👀 **화면 미리보기 설명**

> > 이 기능은 마치 AI와 나눈 대화를 라디오 방송처럼 만들어주는 것과 같아요. 이동 중이거나 눈으로 글을 읽기 힘들 때, 귀로 편하게 핵심 내용을 들을 수 있죠. 목소리 톤이나 스크립트 수정도 가능해서 나만의 맞춤형 오디오 브리핑을 만들 수 있어요.

-----

✅ **완료 점검**

  - [ ] 헤드폰 아이콘을 눌러 오디오가 생성되었나요?
  - [ ] 재생(▶) 버튼을 눌렀을 때, 대화 요약 내용이 음성으로 잘 나오나요?

-----

⚠️ **흔한 실수 & 해결법**

  - **실수**: 대화 내용이 너무 짧거나 없어서 오디오 브리프가 안 만들어져요.
  - **해결**: 최소 3\~4번 이상 AI와 의미 있는 대화를 나눈 후에 오디오 브리프 생성을 시도해보세요.

-----

💡 **응용/고급 팁**

  - **팁 1 (회의 준비)**: 회의 자료들을 넣고 AI와 질의응답을 한 뒤, 오디오 브리프를 만들어 회의장으로 이동하면서 들어보세요.
  - **팁 2 (복습)**: 그날 배운 수업 내용을 NotebookLM에 넣고 요약 정리한 뒤, 오디오 브리프로 만들어 잠들기 전에 들어보세요.

## ⚡ 3. 똑똑하게 활용하기: AI 전문가 되기

### 3-1. 시나리오별 활용법 (실습 예제)

🔍 **자세히 보기 (클릭하여 펼치기)**

\<details\>
\<summary\>\<strong\>시나리오 A: 과학 숙제 마스터하기 (학습용)\</strong\>\</summary\>
\<div style="background-color: \#f0f8ff; padding: 15px; border-radius: 5px; margin-top: 10px;"\>

1.  **준비**: '과학 숙제' 노트북 생성 → 교과서 PDF 파일 3개(화산, 지진, 태풍 챕터)를 **소스로 추가**
2.  **요약 노트**: 대화창에 `업로드한 3개 챕터의 핵심 내용을 각각 5줄로 요약해서 노트에 추가해줘.` 라고 질문 ▶ 답변이 마음에 들면 **핀(📌)으로 노트에 저장**
3.  **퀴즈 생성**: 대화창에 `지금까지의 내용을 바탕으로 객관식 퀴즈 3개, 주관식 퀴즈 2개를 만들어줘. 정답은 맨 아래에 따로 표시해줘.` 라고 요청
4.  **결과물**: 중요한 내용만 정리된 요약 노트와 실력 점검용 퀴즈 5개 완성\!

\</div\>
\</details\>

\<details\>
\<summary\>\<strong\>시나리오 B: 회의 내용 브리핑 만들기 (업무용)\</strong\>\</summary\>
\<div style="background-color: \#fff8f0; padding: 15px; border-radius: 5px; margin-top: 10px;"\>

1.  **준비**: '10월 회의 브리핑' 노트북 생성 → 회의록 파일 2개, 참고 웹페이지 1개를 **소스로 추가**
2.  **핵심 파악**: 대화창에 `두 회의록과 웹페이지를 종합해서 이번 프로젝트의 '결정 사항', '다음 할 일(To-do)', '논의가 더 필요한 점'을 정리해줘.` 라고 질문
3.  **오디오 브리핑**: 위 대화 내용을 바탕으로 **'Create audio brief(🎧)'** 버튼 클릭 → 1분짜리 오디오 브리핑 생성
4.  **공유**: 노트북 오른쪽 위 **'Share'** 버튼 클릭 → **'Share access'** 권한을 \*\*'Viewer'\*\*로 설정하고 링크 복사 → 팀원에게 전달

\</div\>
\</details\>

\<div style="background-color: \#f0fff0; padding: 15px; border-radius: 5px; margin-top: 10px;"\>

1.  **준비**: 'AI 윤리 연구' 노트북 생성 → 관련 논문 PDF 2개를 **소스로 추가**
2.  **비교 분석**: 대화창에 `두 논문의 '공통 주장'과 '핵심적인 차이점'을 표 형태로 비교 분석해줘.` 라고 요청
3.  **인용 포함 초안**: 대화창에 `위 표의 내용을 바탕으로, 두 논문의 관점을 모두 인용한 3문단짜리 요약문을 작성해줘. 모든 문장에 인용 표시를 꼭 달아줘.` 라고 요청
4.  **결과물**: 표로 정리된 비교 분석 자료와 정확한 인용이 포함된 리포트 초안 확보\!
