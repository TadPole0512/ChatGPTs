# 🎙️ NotebookLM Studio 보고서 생성 완벽 가이드

---

## 🧭 요약
NotebookLM Studio에서 AI 음성 대화 형태의 보고서(Audio Overview)를 생성하는 전체 프로세스를 단계별로 안내합니다.

**현재 확실한 사실**: NotebookLM은 Google의 AI 노트 작성 도구로, 2024년부터 Audio Overview 기능 제공 [확인됨]  
**불확실한 부분**: 최신 UI 변경사항은 실시간 확인 필요 [추측 가능]

---

## 📚 근거 & 출처

제공하신 문서에는 NotebookLM 관련 내용이 없어, 제 지식 기반(2025년 1월까지)으로 답변드립니다.

**⚠️ 정보 검증 필요**: 최신 기능은 공식 문서에서 확인을 권장합니다.

## 🧩 단계별 처리 과정

### **1단계: 소스 자료 업로드** 📤

**목표**: 분석할 문서를 NotebookLM에 등록

**입력 자료**:
- PDF, Google Docs, 텍스트 파일, 웹 URL
- 최대 50개 소스, 각 최대 500,000 단어

**실행 방법**:
1. NotebookLM 접속 (notebooklm.google.com)
2. 새 노트북 생성 또는 기존 노트북 열기
3. "Sources" 패널에서 **"+ Add source"** 클릭
4. 업로드 방식 선택:
   - 📁 **Upload**: 로컬 파일 업로드
   - 🔗 **Paste URL**: 웹페이지 링크
   - 📄 **Google Docs/Slides**: Google 드라이브 연동

**출력**: 업로드된 소스 목록 표시

---

### **2단계: Studio (Audio Overview) 생성** 🎧

**목표**: AI 진행자 2명의 대화 형식 오디오 보고서 생성

**실행 방법**:

#### **A. 기본 생성 (자동 모드)**
```
1. 화면 하단 "Notebook guide" 패널 확인
2. "Audio Overview" 섹션에서 
   [Generate] 버튼 클릭
3. 5-10분 대기 (소스 양에 따라 다름)
4. 생성 완료 시 재생 가능
```

**특징**:
- ✅ 자동으로 핵심 내용 추출
- ✅ 2명의 AI 진행자 대화 형식
- ✅ 약 5-15분 분량 오디오

---

#### **B. 커스터마이징 생성 (2024년 12월 업데이트)**

**새로운 "Customize" 기능 활용**:

```
1. [Generate] 옆 [Customize] 버튼 클릭

2. 프롬프트 입력창에서 지시사항 작성:
   예시:
   "초보자를 위해 쉽게 설명해주세요"
   "기술적 세부사항에 집중해주세요"
   "비즈니스 관점에서 분석해주세요"
   "Q&A 형식으로 구성해주세요"

3. [Generate] 클릭

4. 생성 후 만족스럽지 않으면
   다시 [Customize]로 수정 가능
```

**커스터마이징 팁**:
- 📌 청중 지정: "대학생 대상", "경영진 대상"
- 📌 톤 설정: "격식 있게", "캐주얼하게"
- 📌 강조점: "재무 분석 중심", "기술 트렌드 중심"
- 📌 형식 요청: "토론 형식", "인터뷰 형식"

---

### **3단계: 오디오 활용** 🎵

**A. 재생 및 다운로드**
```
✅ 브라우저에서 직접 재생
✅ [Download] 버튼으로 MP3 저장
✅ 재생 속도 조절 (0.8x ~ 1.5x)
✅ 타임스탬프 기능으로 구간 이동
```

**B. 대본(Transcript) 확인**
```
1. 오디오 플레이어 아래 [Show transcript] 클릭
2. 전체 대화 내용 텍스트로 확인
3. 특정 부분 클릭 시 해당 시점으로 이동
4. 대본 복사 가능
```

---

### **4단계: 추가 활용 기능** 🔧

**A. 노트 연동**
- 오디오 들으면서 노트 작성
- 특정 구간을 노트에 인용 가능
- 소스 자료와 교차 참조

**B. 공유 기능**
```
[Share] 버튼 → 링크 생성
- 🔓 Public: 누구나 접근
- 🔒 Restricted: 특정 Google 계정만
```

**C. 여러 버전 생성**
- 같은 소스로 다른 관점의 Audio Overview 생성 가능
- 각각 저장되어 비교 가능

- [ ] 원하는 관점이 반영되었는가?
- [ ] 필요시 [Customize]로 재생성
