## 🔧 영역별 실전 전략

### 1️⃣ 생산성 - 시간 관리

#### Pomodoro 기법

```
25분 집중 → 5분 휴식 (1 Pomodoro)
4 Pomodoro → 15-30분 긴 휴식

도구:
- Focus To-Do (무료, 모바일/PC)
- Pomofocus.io (웹 기반)
```

#### Time Blocking (시간 블록킹)

```
일일 계획 (예시):

09:00-10:30 | 🔴 Deep Work (경조금 신청서 개발)
10:30-10:45 | ☕ Break
10:45-12:00 | 🔴 Deep Work (계속)
12:00-13:00 | 🍱 Lunch
13:00-14:00 | 🟡 회의/이메일
14:00-15:30 | 🔴 Deep Work (테스트 코드 작성)
15:30-15:45 | ☕ Break
15:45-17:00 | 🟢 Shallow Work (문서 정리, 리뷰)
17:00-17:30 | 📝 내일 계획

색상:
🔴 Deep Work (집중 필요)
🟡 Communication (소통)
🟢 Shallow Work (루틴)
```

#### Eisenhower Matrix (우선순위)

| | 긴급 | 긴급하지 않음 |
|---|---|---|
| **중요** | 🔴 즉시 실행<br/>버그 수정, 마감 | 🟢 계획 수립<br/>리팩토링, 학습 |
| **중요하지 않음** | 🟡 위임/최소화<br/>일부 회의 | ⚪ 제거<br/>SNS, 잡담 |

### 2️⃣ 건강 - 루틴 구축

#### 운동 루틴 (주 3회 이상)

```
월/수/금: 근력 운동 (30-40분)
├─ 상체: 푸시업, 덤벨 프레스
├─ 하체: 스쿼트, 런지
└─ 코어: 플랭크, 크런치

화/목: 유산소 (20-30분)
└─ 런닝, 사이클, 수영 중 선택

주말: 산책 또는 등산 (60분)
```

#### 수면 패턴

```
목표: 7-8시간 수면

취침 루틴 (23:00-24:00):
22:00 | 블루라이트 차단 (스마트폰 Night Shift)
22:30 | 가벼운 스트레칭
23:00 | 독서 또는 명상
23:30 | 취침

기상 루틴 (07:00):
07:00 | 알람 즉시 일어나기
07:10 | 물 한 잔
07:20 | 가벼운 스트레칭
07:30 | 샤워
```

#### 영양 균형

```
하루 3끼 균형:
- 단백질: 체중 kg × 1.5g (예: 70kg → 105g)
- 채소: 매 끼니 절반
- 통곡물: 현미, 귀리, 통밀
- 수분: 2L 이상

간식: 견과류, 과일 (가공식품 최소화)
```

### 3️⃣ 재무 - 예산 관리

#### 50/30/20 규칙

```
월 소득: 400만원 (예시)

50% (200만) | 필수 지출
├─ 주거: 100만 (월세/관리비)
├─ 식비: 60만
├─ 교통: 20만
└─ 공과금: 20만

30% (120만) | 선택 지출
├─ 취미: 40만
├─ 외식: 40만
└─ 쇼핑: 40만

20% (80만) | 저축/투자
├─ 비상금: 40만
└─ 투자: 40만
```

#### 비상 자금

```
목표: 월 지출 × 6개월

예시: 300만원 × 6 = 1,800만원

달성 전략:
1개월: 50만원 (시작)
6개월: 300만원 (누적)
36개월: 1,800만원 (목표 달성)
```

#### 투자 다각화

```
포트폴리오 예시:

40% | 안전 자산 (적금, 채권)
30% | 국내 주식 (ETF)
20% | 해외 주식 (S&P 500 ETF)
10% | 기타 (금, 리츠 등)

원칙:
- 월 자동 이체
- 분기별 리밸런싱
- 장기 투자 (5년 이상)
```

### 4️⃣ 관계 - 소통 전략

#### 정기 소통 (주 1회 이상)

```
가족:
└─ 주말 통화 또는 방문

친구:
└─ 월 1회 만남 또는 온라인 게임

동료:
└─ 주 1회 커피 타임

멘토:
└─ 월 1회 조언 구하기
```

#### 감사 표현 (일일 1회)

```
아침: "좋은 아침! 오늘도 화이팅"
점심: "같이 밥 먹어줘서 고마워"
저녁: "오늘 도와줘서 감사합니다"
```

#### 경청 연습

```
5단계 경청:
1. 눈 맞춤
2. 끄덕임
3. 요약 확인 ("그러니까 ~라는 거지?")
4. 공감 ("그 기분 이해해")
5. 조언 (요청 시에만)
```
