## 🎯 5개 도메인 한눈에 보기

### 📋 도메인 1: Claude AI 활용

```
🧠 핵심 시스템 5가지:
├─ Projects: 독립 작업 공간 (100MB Knowledge)
├─ Memory: 자동 개인화 (30개 편집)
├─ Skills: 전문 문서 생성 (docx/pptx/xlsx)
├─ Past Chats: 과거 대화 검색
└─ Computer Use: 파일 자동화

⚡ 즉시 적용 3가지:
1. 프로젝트 1개 생성 (명명: 순서_카테고리_이름)
2. Memory 핵심 3개 추가 (개발 환경, 도메인, 출력 형식)
3. docx 스킬 테스트 (간단한 문서 생성)

📖 상세 문서: 01_claude_ai_guidelines.md
```

### 💻 도메인 2: 프로그래밍 모범 사례

```
🔧 5대 기둥:
├─ 코드 품질: Clean Code, 명명 규칙
├─ 보안: OWASP Top 10, SQL Injection 방지
├─ 성능: N+1 해결, 캐싱, 비동기
├─ 테스트: JUnit 5, 커버리지 80%
└─ 문서화: JavaDoc, README

⚡ 즉시 적용 3가지:
1. 메소드 30줄 이하로 제한
2. PreparedStatement 사용 (SQL Injection 방지)
3. 단위 테스트 작성 (최소 1개)

📖 상세 문서: 02_programming_guidelines.md
```

### 📱 도메인 3: 앱 사용법 최적화

```
🛠️ 핵심 도구 3가지:
├─ IntelliJ IDEA: 단축키 10개 + 디버깅
├─ GitHub Copilot: AI 코딩 어시스턴트
└─ Git/GitLab: 브랜치 전략 + PR

⚡ 즉시 적용 3가지:
1. IntelliJ 단축키 5개 암기 (Ctrl+N, Ctrl+B, F8, F7, Alt+Insert)
2. Copilot Chat 명령 3개 사용 (/explain, /fix, /tests)
3. Git 커밋 메시지 컨벤션 준수 (feat/fix/docs)

📖 상세 문서: 03_app_usage_guidelines.md
```

### 🏠 도메인 4: 생활 최적화

```
🌟 4대 영역:
├─ 생산성: Pomodoro + Time Blocking
├─ 건강: 주 3회 운동 + 7-8시간 수면
├─ 재무: 50/30/20 예산 규칙
└─ 관계: 주 1회 소통 + 일일 감사

⚡ 즉시 적용 3가지:
1. 일일 Time Blocking (Deep Work 2시간)
2. 주 3회 운동 30분
3. 월 소득의 20% 저축 자동 이체

📖 상세 문서: 04_lifestyle_guidelines.md
```

### 🛍️ 도메인 5: 제품 평가

```
📊 5단계 프로세스:
├─ 1단계: 제품 발견 (검색 + 정보 수집)
├─ 2단계: 초기 평가 (필수 기준 체크)
├─ 3단계: 상세 분석 (5가지 기준 점수)
├─ 4단계: ROI 분석 (투자 대비 효과)
└─ 5단계: 구매 결정 (최종 체크리스트)

⚡ 즉시 적용 3가지:
1. 필수 기준 3가지 정의 (기능/예산/AS)
2. 비교 테이블 작성 (3개 제품)
3. ROI 계산 (100% 이상 시 구매)

📖 상세 문서: 05_product_evaluation_guidelines.md
```
