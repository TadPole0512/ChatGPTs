### 3️⃣ 상세 분석 (5가지 기준)

#### 평가 기준 및 가중치

| 기준 | 설명 | 가중치 | 평가 요소 |
|------|------|--------|----------|
| **기능성** | 필요 기능 충족도 | 35% | • 핵심 기능<br>• 확장성<br>• 사용성 |
| **가격** | 가성비 | 25% | • 절대 가격<br>• 경쟁 제품 대비<br>• 프로모션 |
| **품질** | 내구성, 안정성 | 20% | • 재질<br>• 제조 품질<br>• 리뷰 평점 |
| **지속가능성** | 환경, 장기 사용 | 10% | • 에너지 효율<br>• 재활용<br>• 업데이트 지원 |
| **고객 지원** | AS, 문의 응대 | 10% | • AS 센터<br>• 응답 속도<br>• 보증 기간 |

#### 비교 테이블 작성

**예시: 노트북 3종 비교**

| 기준 | 제품 A | 제품 B | 제품 C |
|------|--------|--------|--------|
| **기능성** | ⭐⭐⭐⭐⭐<br/>i7, 16GB, SSD 512GB | ⭐⭐⭐⭐<br/>i5, 16GB, SSD 256GB | ⭐⭐⭐<br/>i5, 8GB, SSD 256GB |
| **가격** | ⭐⭐⭐<br/>120만원 | ⭐⭐⭐⭐⭐<br/>90만원 | ⭐⭐⭐⭐<br/>70만원 |
| **품질** | ⭐⭐⭐⭐<br/>알루미늄, 평점 4.5 | ⭐⭐⭐⭐<br/>플라스틱, 평점 4.2 | ⭐⭐⭐⭐⭐<br/>메탈, 평점 4.7 |
| **지속가능성** | ⭐⭐⭐⭐<br/>에너지 A등급 | ⭐⭐⭐<br/>B등급 | ⭐⭐⭐⭐⭐<br/>A+등급 |
| **고객 지원** | ⭐⭐⭐⭐⭐<br/>3년 보증, 전국 AS | ⭐⭐⭐<br/>1년 보증, 서울만 AS | ⭐⭐⭐⭐<br/>2년 보증, 주요 도시 |
| **종합 점수** | **4.2** | **3.9** | **4.0** |

#### 점수 계산 방법

```
총점 = (기능성 × 0.35) + (가격 × 0.25) + (품질 × 0.20) + 
       (지속가능성 × 0.10) + (고객 지원 × 0.10)

제품 A:
= (5 × 0.35) + (3 × 0.25) + (4 × 0.20) + (4 × 0.10) + (5 × 0.10)
= 1.75 + 0.75 + 0.80 + 0.40 + 0.50
= 4.2
```

### 4️⃣ ROI 분석

#### 투자 대비 효과 계산

```
ROI = (효과 - 비용) / 비용 × 100

예시: 노트북 구매

비용:
- 제품 가격: 120만원
- 액세서리: 10만원
- 총 비용: 130만원

효과 (5년 사용 가정):
- 업무 효율 증대: 월 5시간 절약 × 시급 2만원 × 60개월 = 600만원
- 부업 수입: 월 10만원 × 60개월 = 600만원
- 총 효과: 1,200만원

ROI = (1,200만 - 130만) / 130만 × 100 = 823%
```

#### 의사결정 기준

```
ROI > 100%: 구매 추천
ROI 50-100%: 신중 검토
ROI < 50%: 재고려 필요
ROI < 0%: 구매 비추천
```

### 5️⃣ 구매 결정

#### 최종 체크리스트

```
- [ ] 초기 평가 통과
- [ ] 5가지 기준 점수 계산
- [ ] ROI 100% 이상
- [ ] 리뷰 확인 (최소 50개 이상)
- [ ] 가격 비교 (3곳 이상)
- [ ] 프로모션/할인 확인
- [ ] 반품 정책 확인
- [ ] AS 센터 위치 확인
- [ ] 예산 최종 확인
- [ ] 배우자/가족 동의 (고가 제품)
```

#### 구매 타이밍

```
즉시 구매:
- 필수 제품 (업무용 노트북 고장 등)
- 프로모션 종료 임박
- 재고 한정

대기 추천:
- 신제품 출시 예정 (1-2개월 내)
- 대규모 세일 시즌 (블랙프라이데이 등)
- 가격 하락 추세
```

## 💡 구매 실패 방지 팁

### 충동 구매 방지

```
24시간 규칙:
└─ 고가 제품은 24시간 대기 후 재검토

위시리스트 활용:
└─ 즉시 구매 X → 위시리스트 추가 → 1주일 후 재평가

질문 3가지:
1. 정말 필요한가?
2. 지금 당장 필요한가?
3. 대체 가능한가?
```

### 마케팅 트릭 인지

```
❌ "한정 수량!" → 압박 전술
❌ "최저가 보장!" → 실제 확인 필요
❌ "99,900원" → 10만원이라고 생각하기

✅ 실제 필요성 중심 판단
✅ 최소 3곳 가격 비교
✅ 리뷰 확인 (평점 + 내용)
```
