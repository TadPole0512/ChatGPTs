## 💡 핵심 내용 미리보기

### Claude Skills가 할 수 있는 10가지:

1. **📄 문서 자동화** - PPT, Excel, Word, PDF
2. **📊 데이터 분석** - 월간 보고서, 트렌드 분석
3. **✍️ 콘텐츠 생성** - 블로그, SNS, 이메일
4. **🎯 마케팅** - A/B 테스트, 브랜드 가이드 적용
5. **👥 HR** - 채용, 면접, 온보딩
6. **💻 개발** - API 문서, 코드 리뷰
7. **📋 프로젝트 관리** - 계획, 진도, 위험 분석
8. **🏢 법무** - 계약서, 정책 문서
9. **🎓 교육** - 교재, 퀴즈, 학습 경로
10. **🔍 데이터 검색** - 코드 분석, 문서 분류

### 첫 스킬을 만드는 과정 (5분 요약):

```
1. 폴더 생성: my-first-skill
   └── SKILL.md 파일 생성

2. SKILL.md 파일 내용:
   - name: skill-name
   - description: 상세한 설명
   - 본문: 지침, 형식, 예시

3. ZIP 파일 생성
   → my-first-skill.zip

4. Claude.ai 업로드
   Settings → Capabilities → Skills

5. 테스트 및 사용!
```
