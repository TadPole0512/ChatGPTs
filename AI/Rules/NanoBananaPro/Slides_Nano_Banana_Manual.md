# 🎯 Google Slides + Nano Banana Pro 실무 완벽 매뉴얼 (v2.0)

## 📌 Executive Summary (요약)

- 🟢 **Nano Banana Pro (Gemini 3 Pro Image)**는 2025년 11월 20일 Google Workspace에 순차 출시 완료. Google Slides, Vids, Gemini 앱, NotebookLM에서 사용 가능

- 🟡 **Nano Banana Pro 텍스트 렌더링**: 이전 버전 대비 70% 정확도 향상 (추측, Google 공식 수치 미공개)

### 섹션 1️⃣ Google Slides 기본 조작법 (Professional 기초)

#### 1-1. 슬라이드 계층 구조 이해하기

**무엇인가:**
Google Slides 프레젠테이션은 4단계 계층 구조로 구성됩니다:

```
┌─ Presentation (프레젠테이션)
│  │
│  └─ Theme (테마/마스터)
│     │
│     ├─ Layout (레이아웃 1: 제목 슬라이드)
│     ├─ Layout (레이아웃 2: 제목+본문)
│     └─ Layout (레이아웃 3: 빈 슬라이드)
│        │
│        └─ Slide 1, Slide 2, ... (개별 슬라이드)
│           │
│           └─ Page Elements (텍스트박스, 이미지, 도형)
```

**어디에 사용:**
- **테마 수정**: 모든 슬라이드에 일괄 적용 (색, 폰트, 배경)
- **레이아웃 설정**: 슬라이드 타입별 템플릿 (표지/내용/섹션)
- **개별 슬라이드 편집**: 텍스트, 이미지, 도형 추가/수정

**실무 팁:**
- 테마 변경 후 개별 슬라이드 색/폰트 수정 = 일관성 깨짐 위험 ⚠️
- 해결책: 테마에서 한 번만 설정, 개별 슬라이드는 수정 금지
- 예외: 특정 슬라이드만 다른 디자인 원할 때 → 테마 빌더에서 새 레이아웃 추가

**✅ 5단계 빠른 시작**
1. `drive.google.com` 열기
2. **+ 새로 만들기** → **프레젠테이션** 클릭
3. **슬라이드** > **템플릿 갤러리** → 템플릿 선택
4. 텍스트 상자 클릭 후 제목/내용 입력
5. **슬라이드** > **새 슬라이드** (또는 **Ctrl+M**) 추가

---

#### 1-2. 테마 빌더 (Theme Builder) 마스터 가이드

**정의:**
Google Slides의 "테마 빌더" (이전 이름: 마스터 슬라이드)는 모든 슬라이드의 **시각 규칙**을 중앙에서 관리하는 도구입니다. 2025년부터 "마스터" 용어 대신 "테마 빌더"로 공식명칭 변경.

**관리 가능 항목:**
- **색상 팔레트**: Accent 1~6 (회사 브랜드 컬러)
- **기본 폰트**: 제목, 본문, 서식 지정 (전사 표준 폰트)
- **배경**: 이미지 또는 그라데이션
- **레이아웃별 자리표시자**: 제목/본문 위치, 로고 삽입 위치

**어디에 접근:**
**보기 > 테마 빌더** (또는 오른쪽 사이드바 > "테마 빌더" 탭)

**실무 워크플로우:**

| 단계 | 작업 | 메뉴 경로 | 결과 |
|---|---|---|---|
| 1단계 | 마스터 슬라이드 선택 | 왼쪽 패널 > 맨 위 항목 | 전체 설정 화면 |
| 2단계 | 색상 테마 정의 | 오른쪽 **색상** 버튼 | Accent 1~6 설정 |
| 3단계 | 폰트 지정 | 오른쪽 **폰트** 버튼 | 제목/본문 폰트 선택 |
| 4단계 | 배경 설정 | 오른쪽 **배경** 버튼 | 이미지/색상 선택 |
| 5단계 | 개별 레이아웃 편집 | 왼쪽 > 레이아웃 이름 | 각 타입별 위치 조정 |
| 6단계 | 완료 | 오른쪽 **X** 또는 **완료** | 테마 빌더 종료 |

**✅ 테마 빌더 5단계 실행**
```
1. 프레젠테이션 열기
2. 보기 > 테마 빌더 클릭
3. 왼쪽에서 "Master" (맨 위) 선택
4. 오른쪽 "색상" 버튼 > "테마 컬러 선택"
5. X 또는 "완료"로 테마 빌더 종료
```

**주의사항:**
- 마스터 변경 후 개별 슬라이드에서 다시 수정하면 전체 일관성 깨짐
- 회사 로고를 마스터에 삽입하면 모든 슬라이드에 자동 출현
- 폰트는 웹에서 지원되는 Google Fonts만 추천 (호환성)

**대체 경로 (기존 테마 가져오기):**
**슬라이드** > **테마 변경** > **테마 가져오기** > 기존 프레젠테이션 선택
