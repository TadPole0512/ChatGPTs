# 🔌 전자제품/가전제품 활용 극대화 가이드라인 설계

---

## 🧭 사고 과정 (Thinking Process)

이 질문에 답하기 위해 다음 공식들을 조합하여 접근했습니다:

### 적용한 공식들:
1. **문제 재정의 알고리즘 (PR)**: 단순 "사용법"이 아닌 "활용 극대화"로 관점 전환
2. **다차원 분석 프레임워크 (MDA)**: 시간적(구매-설치-사용-유지보수), 공간적(가정-직장-이동), 계층적(개별기능-통합활용-생태계) 차원 분석
3. **복잡성 해결 매트릭스 (CS)**: 제품군별 특성을 하위 시스템으로 분해
4. **통합 지혜 공식 (IW)**: 지식(제품 스펙) + 이해(작동 원리) + 실행(실제 활용) + 윤리(안전/환경)

### 핵심 인사이트:
- **관찰(O)**: 대부분의 사용자는 제품 기능의 20-30%만 활용 (Pareto의 법칙)
- **연결(C)**: 제품 간 연동, 생활 패턴과의 통합이 핵심
- **패턴(P)**: 구매 → 초기 설정 → 일상 사용 → 문제 해결 → 업그레이드 주기 존재
- **종합(S)**: 개별 제품이 아닌 "생활 시스템"으로 접근 필요

def basic_verification(product_info):
    """모든 답변에 필수로 포함되어야 할 검증"""
    
    checklist = {
        "1. 출처 명시": {
            "요구사항": "공식 웹사이트, 매뉴얼, 인증된 리뷰어",
            "형식": "[출처명](URL) - 확인일자: YYYY-MM-DD",
            "예시": "[삼성전자 공식](https://samsung.com/model123) - 2025-09-30"

        "2. 실물 확인": {
            "방법": ["제품 사진", "설정 화면 캡처", "모델명 스티커"],

            "대안": "공식 이미지 사용 시 명시"
        },
        
        "3. 수치 검증": {
            "모든_숫자": "출처 + 측정 방법 + 단위",

    return checklist
