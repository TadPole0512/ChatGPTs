## ✅ B. 6차원 품질 평가

### 평가 기준 (총 100점)

| 차원 | 가중치 | 합격 기준 | 자동 검증 |
|------|--------|-----------|-----------|
| 정확성 | 25% | ≥70점 | URL/코드/수식 검증 |
| 완성도 | 20% | ≥70점 | 체크리스트 확인 |
| 명확성 | 15% | ≥60점 | 가독성 점수 (Flesch-Kincaid) |
| 실행가능성 | 20% | ≥70점 | 코드 실행 테스트 |
| 안전성 | 10% | ≥60점 | 민감정보/위험 명령어 탐지 |
| 효율성 | 10% | ≥60점 | 복잡도 분석 |

### 자동 검증 스크립트
```python
# 품질 검증 파이프라인
def verify_quality(content: str) -> dict:
    evaluations = [
        verify_accuracy(content),     # URL, 코드, 수식
        check_completeness(content),  # 체크리스트
        assess_clarity(content),      # 가독성, 구조
        validate_actionability(content), # 실행 가능
        check_safety(content),        # 보안, 위험
        assess_efficiency(content)    # 복잡도, 길이
    ]
    
    total = sum(e['score'] * e['weight'] for e in evaluations)
    
    return {
        'score': total,
        'grade': 'A' if total>=90 else 'B' if total>=80 else 'C',
        'pass': total >= 70
    }
```

## 🎯 핵심 요약

**협업 5단계**: 스코핑 → 분해 → 구현 → 검증 → 인계

**품질 6차원**: 정확성(25%) + 완성도(20%) + 명확성(15%) + 실행가능성(20%) + 안전성(10%) + 효율성(10%)

**자동화**: 컨텍스트 압축 + 체크포인트 + 품질 검증 + 자동 개선

**목표**: 복잡한 프로젝트를 체계적으로 완수하고, 모든 답변의 품질을 객관적으로 보증

---

**적용 시기**: 3회 이상 대화 필요한 모든 작업 (개발, 문서화, 설계, 분석 등)

**예상 효과**: 완료율 ≥95%, 재작업률 ≤10%, 사용자 만족도 "기대 충족" 이상
