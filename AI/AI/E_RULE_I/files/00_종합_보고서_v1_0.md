최종 인계:

### 3️⃣ 품질 보증 프레임워크 지침

**파일명**: `Quality_Assurance_Framework_Detailed_v1_0.md` (24KB)

**핵심 내용**:
- **6차원 평가**: 정확성(25%) + 완성도(20%) + 명확성(15%) + 실행가능성(20%) + 안전성(10%) + 효율성(10%)
- **자동 검증 스크립트**: URL/코드/수식 자동 체크
- **자가 개선 루프**: 평가 → 개선 제안 → 자동 수정 → 재평가
- **품질 보고서**: Markdown 형식 자동 생성

**적용 시기**: 모든 AI 생성 콘텐츠

**평가 프로세스**:
```python
# 자동 품질 검증
def verify_quality(content: str) -> dict:
    evaluations = [
        verify_accuracy(content),      # 25%
        check_completeness(content),   # 20%
        assess_clarity(content),       # 15%
        validate_actionability(content), # 20%
        check_safety(content),         # 10%
        assess_efficiency(content)     # 10%
    ]
    
    total = sum(e['score'] * e['weight'] for e in evaluations)
    
    return {
        'score': total,
        'grade': 'A' if total>=90 else 'B' if total>=80 else 'C',
        'pass': total >= 70  # 합격 기준
    }
```
