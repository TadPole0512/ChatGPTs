
# ✅ GPT 분석 체크리스트 도우미

이 체크리스트는 Custom GPT를 설계하거나 점검할 때 구조적 문제를 진단하고 개선 방향을 정리하기 위한 가이드입니다. 각 항목에 대해 체크(✅ / ❌) 또는 직접 메모를 작성하세요.

| 항목       | 질문                                                       | 체크 | 메모 |
|------------|------------------------------------------------------------|------|------|
| 🎯 목적    | GPT의 최종 목적이 한 문장으로 명확히 설명되어 있는가?       |      |      |
| 🧠 역할    | GPT의 행동 방향이 구체적인 역할로 설정되어 있는가?         |      |      |
| 💬 말투/톤 | 말투, 응답 스타일이 명시되어 있거나 일관성이 있는가?      |      |      |
